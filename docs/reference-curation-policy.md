# 学習用参考答案の整備方針

## 目的

公式模範解答が公表されない土木2次の記述問題について、民間解答例をそのまま保存・転載せず、公式問題文と一般的な施工管理知識に基づく学習用参考答案へ整理する。

## 用語

- `official_answer`: 公式に公表された正答又は正答肢。
- `third_party_reference`: TAC、地域開発研究所、日建学院、CIC、過去問サイト等の民間解答例や総評。
- `curated_reference`: 民間解答例等を参照候補として確認し、論点・キーワード・注意点を抽出したうえで、独自に作成した学習用参考答案。
- `ai_reference`: AIだけで作成した未検証の参考答案。

## 保存してよいもの

- 公式問題PDFのURL、確認日、対象年度、対象問題。
- 民間解答例のサイト名、URL、確認日、参照種別。
- 自分の言葉で整理した論点、キーワード、採点観点、注意点。
- 独自に作成した学習用参考答案。

## 保存しないもの

- 民間解答例の本文コピー。
- 民間解答例の文章を言い換えただけの答案。
- 有料PDF、書籍、講座資料等の本文。
- 公開アプリ、Git、GAS、Spreadsheetに投入する目的の第三者原文。

## 画面表示

表示名は「模範解答」ではなく「学習用参考答案」又は「参考答案」とする。

必ず次の趣旨を表示する。

> この参考答案は公式解答ではありません。公開問題、民間解答例、一般的な施工管理知識を参考に、学習用に独自作成したものです。

## 採点扱い

- 公式正答肢がある問題は `deterministic` として扱える。
- 記述問題は `sourceQuality=reference_only` とし、AI採点は公式採点として扱わない。
- 正答キー未確認の穴埋め・数値・選択問題は、正答キーが確認できるまで deterministic 採点にしない。
